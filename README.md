# JunKai Chen Personal Website

Static personal website for general internship and early-career job applications.

## GitHub Pages

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to **Pages**.
4. Set **Source** to `Deploy from a branch`.
5. Select `main` and `/ (root)`.

The site is built with plain HTML, CSS, and JavaScript, so no build step is required.
